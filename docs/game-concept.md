# 游戏概念文档

## 游戏名称（暂定）
**《命运卡牌》** / **Card of Fate**

## 核心概念
一款轻量级卡牌策略闯关游戏，玩家通过构建卡组、策略出牌，战胜一系列敌人并最终通关。

## 设计哲学
- **简单易上手**：规则清晰，新手指引完善
- **策略深度**：卡牌组合、出牌顺序、资源管理
- **快节奏**：单局战斗 5-10 分钟
- **高回玩性**：不同卡组搭配，多种通关路线
- **无美术依赖**：使用 CSS/符号实现视觉效果

## 核心玩法循环
```
选择关卡 → 战斗（出牌策略）→ 获得奖励/新卡牌 → 强化卡组 → 下一关
```

## 目标玩家
- 喜欢策略思考的玩家
- 碎片化时间娱乐需求
- 卡牌游戏入门者

## 独特卖点
1. **动态难度**：根据玩家表现调整敌人强度
2. **卡牌融合**：特定卡牌组合可产生新效果
3. **元游戏**：通关后解锁新模式/卡牌
