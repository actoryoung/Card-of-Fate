# 战斗机制

## 战斗流程

### 回合结构
```
玩家回合：
  1. 开始阶段：触发回合开始效果
  2. 抽牌阶段：抽 5 张牌（可被技能影响）
  3. 出牌阶段：使用能量打出卡牌
  4. 弃牌阶段：未使用的牌进入弃牌堆
  5. 结束阶段：触发回合结束效果

敌人回合：
  1. 意图展示：显示下回合行动
  2. 执行行动：攻击/防御/技能
  3. 状态结算：持续伤害/护甲计算
```

## 核心数值

### 玩家属性
- **生命值 (HP)**：默认 80
- **能量 (Energy)**：每回合回复 3 点
- **护甲 (Armor)**：抵挡伤害，回合结束不消失
- **抽牌数**：默认 5 张/回合

### 敌人属性
- **生命值 (HP)**：因敌人而异
- **攻击力 (ATK)**：基础伤害值
- **护甲 (Armor)**：部分敌人拥有
- **特殊能力**：独特机制

## 伤害计算
```
最终伤害 = (基础伤害 + 加成) × 倍率 - 敌人护甲
最小伤害 = 1（至少造成 1 点伤害）
```

## 状态效果
| 状态 | 效果 | 持续 |
|------|------|------|
| 中毒 | 每回合失去固定生命 | 3回合 |
| 燃烧 | 每回合百分比生命伤害 | 2回合 |
| 虚弱 | 造成的伤害减少25% | 2回合 |
| 易伤 | 受到的伤害增加50% | 2回合 |
| 缴械 | 无法攻击 | 1回合 |

## 特殊机制
- **连击系统**：连续使用同类型卡牌获得加成
- **完美时机**：在敌人攻击前打出防御卡获得额外护甲
- **终结技**：能量满时可以发动强力技能
