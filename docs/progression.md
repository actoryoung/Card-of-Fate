# 闯关与成长系统

## 关卡结构

### 区域设计
```
第一章：新手教程 (1-3关)
  ├── 关卡1：基础战斗教学
  ├── 关卡2：卡牌组合教学
  └── 关卡3：BOSS战（弱）

第二章：森林遗迹 (4-7关)
  ├── 随机遭遇战
  ├── 精英敌人
  ├── 休息点（回复生命/改造卡组）
  └── BOSS战

第三章：...（后续扩展）
```

### 关卡类型
1. **普通战**：常规敌人，获得金币/卡牌
2. **精英战**：强力敌人，稀有奖励
3. **休息点**：回复生命 / 移除卡牌 / 升级卡牌
4. **事件**：随机选择，风险与收益
5. **BOSS战**：区域守关，必须击败

## 奖励系统

### 战斗奖励
- **金币**：用于商店购买卡牌/道具
- **卡牌**：从 3 张随机卡牌中选择 1 张
- **道具**：一次性使用物品
- **经验**：解锁新内容

### 商店系统
```
┌─────────────────┐
│  卡牌商店       │
│  - 普通卡：50金币  │
│  - 稀有卡：100金币 │
│  - 随机卡包：150金币│
└─────────────────┘
```

## 卡牌强化

### 升级机制
- **选择提升**：从 3 张卡牌中选择 1 张升级
- **升级效果**：伤害+3、费用-1、增加附加效果
- **移除机制**：从卡组中移除不需要的卡牌

## 道具系统（可扩展）
- **生命药水**：回复 20 点生命
- **能量药水**：战斗中 +2 能量
- **抽卡卷轴**：战斗中抽 2 张额外卡牌
- **护盾图腾**：战斗开始获得 10 点护甲

## 存档系统
- **自动存档**：每关结束后
- **进度保存**：当前区域、卡组、生命值
- **解锁保存**：永久解锁的内容
