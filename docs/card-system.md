# 卡牌系统设计

## 卡牌类型

### 1. 攻击卡 (Attack)
直接造成伤害
- **基础攻击**：造成 6 点伤害
- **重击**：消耗 2 能量，造成 15 点伤害
- **连斩**：造成 3 点伤害，可连续使用

### 2. 防御卡 (Defense)
获得护甲或减少伤害
- **铁壁**：获得 8 点护甲
- **闪避**：本回合获得 50% 闪避率
- **反弹**：将受到伤害的 50% 反射给敌人

### 3. 技能卡 (Skill)
特殊效果和战术
- **抽卡**：抽 2 张牌
- **充能**：获得 2 点能量
- **虚弱**：敌人下回合攻击力减半

### 4. 状态卡 (Status)
持续效果
- **中毒**：敌人每回合失去 3 点生命（持续 3 回合）
- **燃烧**：敌人每回合失去 5 点生命（持续 2 回合）
- **护盾**：持续 3 回合的护甲

```json
  "icon": "⚔️"
}
```

## 卡组规则
- **起始卡组**：10 张固定卡牌
- **卡组上限**：20 张
- **每回合抽牌**：5 张
- **手牌上限**：10 张
- **能量上限**：3 点/回合

## 卡牌稀有度
- **普通 (Common)**：基础卡牌，大量出现
- **稀有 (Rare)**：强力效果，中等出现
- **史诗 (Epic)**：独特机制，较少出现
- **传说 (Legendary)**：改变战局，极稀有
