# 命运河牌 (Card of Fate)

一款轻量级卡牌策略闯关游戏，使用纯 HTML/CSS/JavaScript 开发。

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm start
```

浏览器会自动打开 `http://localhost:8080`

## 项目结构

```
new_try/
├── docs/           # 游戏设计文档
├── data/           # 卡牌、敌人、关卡数据
├── src/            # 源代码
│   ├── core/       # 核心逻辑
│   ├── ui/         # 界面渲染
│   ├── utils/      # 工具函数
│   ├── styles/     # CSS 样式
│   └── main.js     # 入口文件
└── index.html      # 主页面
```

## 游戏特色

- 无需游戏引擎，浏览器直接运行
- 纯 CSS 实现视觉效果
- 策略性卡牌战斗系统
- 章节式闯关模式
- 可扩展的卡牌和敌人系统
